Learn Lean

Thu, 2025-07-17

Learn how to use an advanced programming language to solve mathematical puzzles and verify them.

  • Contents
  • Price
  • How to find us

Contents

Plan:

  • Introduction to the mathematical proof assistant and programming language Lean4. It can be used to implement software, but is currently very popular for formalizing mathematics.
  • Introduction Mathlib, the standard library for Lean. It contains a lot of useful definitions and theorems from mathematics, which you can use to solve problems.
  • Practice Lean by solving and analyzing riddles, which will be posted on GitHub.

Price

The workshop and presentation itself is free.

Use the link to the Signal chat if you want to join dinner around Vrijdagmarkt before the workshop (optional).
